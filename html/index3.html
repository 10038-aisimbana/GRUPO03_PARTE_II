En el desarrollo web, el HTML DOM avanzado ofrece una serie de técnicas y características que permiten a los desarrolladores crear sitios web más interactivos y personalizados. Estas funcionalidades incluyen la capacidad de agregar animaciones, controlar eventos, manipular nodos y trabajar con colecciones de elementos.

HTML DOM avanzado proporciona herramientas y técnicas para crear sitios web más interactivos y personalizados. Con animaciones, eventos, manipulación de nodos, colecciones y navegación, los desarrolladores tienen la capacidad de mejorar la experiencia del usuario y hacer que las páginas web sean más dinámicas y atractivas.

Las animaciones DOM permiten agregar efectos visuales y de transición a los elementos de una página web, lo que mejora la experiencia del usuario y hace que la página sea más atractiva.

Los eventos DOM son acciones o interacciones que ocurren en una página web, como hacer clic en un botón, mover el cursor sobre un elemento o desplazarse por la página. Estos eventos se pueden detectar y manejar utilizando JavaScript, lo que permite realizar acciones específicas en respuesta a las interacciones del usuario.

Los nodos DOM son los elementos individuales que componen una página web, como etiquetas HTML, texto y atributos. Los nodos se pueden acceder y manipular mediante JavaScript, lo que permite modificar la estructura y el contenido de una página de forma dinámica

Las colecciones DOM son conjuntos de elementos relacionados que se pueden acceder y manipular como un grupo. Esto es útil cuando se desea realizar operaciones en varios elementos a la vez, como cambiar el estilo de todos los elementos con una clase específica.

La navegación DOM se refiere a la capacidad de desplazarse y acceder a los elementos de una página web de manera eficiente.

Esto implica la capacidad de moverse por la estructura jerárquica del DOM y acceder a los nodos deseados.


Animaciones DOM

Las animaciones DOM son una técnica que permite agregar efectos visuales y transiciones a los elementos de una página web, creando una experiencia más dinámica y atractiva para los usuarios. Estas animaciones se logran modificando las propiedades de los elementos HTML a lo largo del tiempo utilizando JavaScript.

Es importante tener en cuenta que las animaciones DOM pueden afectar el rendimiento de una página web si no se manejan correctamente. Se recomienda optimizar las animaciones utilizando técnicas como el uso de transformaciones 3D (que aprovechan la aceleración por hardware) y limitar el número de animaciones simultáneas para garantizar una experiencia fluida y sin problemas.

Ejemplo

function iniciarAnimacion() {
    var balon = document.getElementById('balon');
    var posicion = 0;
    var direccion = 1;
    var velocidad = 2;

    function animar() {
    posicion += velocidad * direccion;

    if (posicion > 200 || posicion < 0) {
    direccion *= -1;
    }

    balon.style.left = posicion + 'px';

    idAnimacion = requestAnimationFrame(animar);
    }

    animar();
    }

Eventos DOM

Existen numerosos eventos DOM disponibles en JavaScript para interactuar con los elementos HTML y realizar acciones en respuesta a diferentes tipos de eventos. A continuación se muestra una lista de algunos de los eventos DOM más comunes:

    click: Se dispara cuando se hace clic en un elemento.

    onmouseover: Se dispara cuando el puntero del mouse se mueve sobre un elemento.

    change: Se dispara cuando el valor de un elemento cambia (utilizado principalmente en elementos de formulario).

    keydown: Se dispara cuando se presiona una tecla.

    keyup: Se dispara cuando se suelta una tecla.

    keypress: Se dispara cuando se presiona una tecla que produce un carácter.

    focus: Se dispara cuando un elemento recibe el foco (seleccionado para entrada de datos).

    blur: Se dispara cuando un elemento pierde el foco.

    submit: Se dispara cuando se envía un formulario.

    scroll: Se dispara cuando se desplaza la barra de desplazamiento de un elemento.

    load: Se dispara cuando se completa la carga de un recurso (como una imagen o un documento).

    unload: Se dispara cuando se descarga una página o se navega lejos de ella.

    resize: Se dispara cuando se cambia el tamaño de la ventana del navegador.

    contextmenu: Se dispara cuando se hace clic con el botón derecho del ratón para mostrar el menú contextual.

Estos son solo algunos ejemplos de los eventos DOM disponibles. Existen muchos más eventos que se pueden utilizar en diferentes situaciones, y también es posible crear eventos personalizados.

EventListener DOM

Los event listeners (escuchadores de eventos) son funciones o bloques de código que se asocian a un elemento del DOM (Document Object Model) para escuchar y responder a eventos específicos que ocurren en ese elemento.

Los event listeners se agregan a un elemento utilizando el método addEventListener() disponible en los elementos del DOM. Toma dos argumentos: el nombre del evento que se quiere escuchar (por ejemplo, "click", "mouseover", "keydown", etc.) y una función que se ejecutará cuando ocurra el evento.

Ejemplo

document.addEventListener("DOMContentLoaded", function() {
    var taskInput = document.getElementById("taskInput");
    var addButton = document.getElementById("addButton");
    var taskList = document.getElementById("taskList");

    addButton.addEventListener("click", function() {
    var taskText = taskInput.value;

    if (taskText.trim() !== "") {
    var newTask = document.createElement("li");
    newTask.innerText = taskText;

    newTask.addEventListener("click", function() {
    newTask.classList.toggle("completed");
    });

    taskList.appendChild(newTask);
    taskInput.value = "";
    }
    });
    });

Navegación DOM

La navegación DOM, también conocida como DOM navigation en inglés, se refiere al proceso de recorrer y acceder a diferentes elementos en la estructura del DOM (Document Object Model) utilizando métodos y propiedades proporcionados por JavaScript.

El DOM representa la estructura de un documento HTML o XML como un árbol de objetos, donde cada elemento del documento (como etiquetas

,

, etc.) se convierte en un nodo en el árbol. La navegación DOM implica acceder y manipular estos nodos para realizar diversas operaciones, como obtener información de los elementos, modificar su contenido, agregar o eliminar elementos, cambiar estilos, entre otras cosas.

La navegación DOM se realiza utilizando métodos y propiedades disponibles en JavaScript, como getElementById, querySelector, childNodes, parentNode, nextSibling, previousSibling, entre otros. Estos métodos y propiedades permiten acceder a elementos específicos, recorrer los nodos hijos o hermanos, y realizar acciones en función de su posición y relación en el árbol DOM.

Ejemplo

En este ejemplo, hemos creado una galería de imágenes utilizando la navegación DOM. La galería consiste en una serie de miniaturas de imágenes, cada una envuelta en un elemento "< div >" con la clase "my-thumbnail". Al hacer clic en una miniatura, se muestra una vista ampliada de la imagen en una ventana modal. La galería se muestra en un contenedor "< div >" con la clase gallery. Cada miniatura tiene un evento onclick que llama a la función "showEnlargedView()", pasando el elemento de la miniatura como argumento.

function showEnlargedView(thumbnail) {
    var enlargedViewContainer = document.querySelector('.enlarged-view-container');
    var enlargedImage = enlargedViewContainer.querySelector('img');

function hideEnlargedView() {

}

Nodos DOM

Los nodos DOM (Document Object Model) son objetos que representan elementos individuales en una estructura de documento HTML o XML. El DOM organiza los elementos del documento en forma de árbol, donde cada elemento, atributo, texto o comentario se representa como un nodo en el árbol.

Existen varios tipos de nodos DOM, y cada uno de ellos tiene diferentes propiedades y métodos que permiten acceder y manipular la estructura y contenido del documento. Algunos de los nodos DOM más comunes son:

    Element Node (Nodo de Elemento): Representa un elemento HTML o XML, como una etiqueta < div >, < p >, < ul >, etc. Los nodos de elemento pueden contener otros nodos, como nodos de texto o nodos de elemento secundarios.

    Text Node (Nodo de Texto): Representa el contenido de texto dentro de un elemento. Por ejemplo, el texto entre las etiquetas < p > y < /p > se representa como un nodo de texto.

    Attribute Node (Nodo de Atributo): Representa un atributo de un elemento HTML o XML. Por ejemplo, el atributo src de una etiqueta < img > se representa como un nodo de atributo.

    Comment Node (Nodo de Comentario): Representa un comentario dentro del documento. Los comentarios se escriben entre < !-- y -- >.

    Document Node (Nodo de Documento): Representa el documento en sí mismo. Es el nodo raíz del árbol DOM.

Estos son solo algunos ejemplos de los tipos de nodos DOM más comunes. Cada nodo tiene propiedades y métodos específicos que permiten acceder y manipular su contenido, atributos, hijos y relaciones con otros nodos en el árbol.El uso de nodos DOM es fundamental para interactuar con la estructura y contenido de un documento HTML o XML utilizando JavaScript u otros lenguajes de programación. Permite acceder a los elementos, modificar su contenido, agregar o eliminar elementos, cambiar estilos y mucho más.

Ejemplo

En este ejemplo, tenemos un botón que, cuando se hace clic, ejecuta la función agregarElemento(). Esta función crea un nuevo elemento de párrafo (< p >) utilizando document.createElement(). Luego, se crea un nodo de texto utilizando document.createTextNode() y se le asigna el contenido del párrafo. Después, se agrega el nodo de texto al párrafo utilizando appendChild(). Finalmente, se agrega el párrafo al contenedor utilizando appendChild() en el elemento con el id "contenedor1".

function agregarElemento() {
    var nuevoParrafo = document.createElement("p");       
    var contenido = document.createTextNode("¡Hola, soy un nuevo párrafo!");
    nuevoParrafo.appendChild(contenido);
    nuevoParrafo.style.color = "green";
    document.getElementById("contenedor1").appendChild(nuevoParrafo);
}

Colecciones DOM

En el contexto de las Collections DOM, "Collections" se refiere a las diferentes colecciones o conjuntos de elementos que se pueden acceder y manipular dentro del DOM. Estas colecciones son objetos que representan grupos de elementos relacionados dentro de un documento.

Algunas de las colecciones DOM más comunes incluyen:

    document:Es la colección principal y representa el documento completo. Proporciona métodos y propiedades para acceder y manipular elementos en todo el documento.

    getElementsByTagName: Es una colección de elementos que se obtienen mediante el nombre de la etiqueta HTML. Por ejemplo, getElementsByTagName("div") devuelve todos los elementos div del documento.

    getElementsByClassName: Es una colección de elementos que se obtienen mediante el nombre de la clase CSS. Por ejemplo, getElementsByClassName("mi-clase") devuelve todos los elementos que tienen la clase "mi-clase".

    getElementById: Es una colección de elementos que se obtienen mediante el ID del elemento. Dado que el ID debe ser único en el documento, esta colección generalmente contiene solo un elemento.

    childNodes: Es una colección de nodos secundarios de un elemento determinado. Los nodos secundarios pueden ser elementos, comentarios, texto, etc.

Estas colecciones proporcionan métodos y propiedades para iterar, acceder y manipular los elementos que contienen. Los desarrolladores web utilizan estas colecciones DOM para realizar tareas como la manipulación del contenido, la modificación de estilos y la interacción con eventos en un documento web.

Ejemplo

function myFunction1() {
      const myCollection = document.getElementsByTagName("p");
      const secondPara = document.getElementsByClassName("secondPara")[0];

      for (let i = 0; i < myCollection.length; i++) {
        myCollection[i].classList.remove("blue");
        myCollection[i].classList.add("highlight");

        if (i === 0) {
          myCollection[i].classList.remove("highlight");
        }

        if (myCollection[i] === secondPara) {
          if (myCollection[i].classList.contains("blue")) {
            myCollection[i].classList.remove("blue");
          } else {
            myCollection[i].classList.add("blue");
          }
        }
      }
    }

Lista de Nodos DOM

Un NodeList es un objeto en el Modelo de Objetos del Documento (DOM) que representa una colección ordenada de nodos. Los nodos en un NodeList se obtienen a través de métodos como getElementsByTagName, getElementsByClassName, querySelectorAll o childNodes, entre otros.

Un NodeList es similar a un array, ya que contiene una lista de elementos, pero no es un array real. A diferencia de un array, un NodeList no tiene todos los métodos y propiedades de un array, como push, pop, slice, etc.

Para acceder a los elementos individuales dentro de un NodeList, se puede utilizar la notación de corchetes y el índice correspondiente. Por ejemplo, nodeList[0] devuelve el primer nodo en el NodeList.

Sin embargo, es importante tener en cuenta que un NodeList no es una estructura de datos estática. Cada vez que se accede a un NodeList, se realiza una consulta en el DOM para obtener los nodos actualizados. Por lo tanto, si se necesita acceder a los mismos nodos varias veces, puede ser más eficiente almacenarlos en una variable y trabajar con esa variable en lugar de realizar consultas repetidas.

Ejemplo

En este ejemplo, el código JavaScript selecciona todos los elementos li dentro de la lista con el identificador myList utilizando document.querySelectorAll('#myList li'), lo que devuelve un NodeList de elementos.

Con este ejemplo, puedes ver cómo se pueden realizar múltiples acciones en los elementos de una lista utilizando un NodeList. Los elementos pares se resaltan con un color de fondo amarillo, mientras que los elementos impares se vuelven negrita y tienen un evento de clic asociado.

var nodeList = document.querySelectorAll('#myList li');


    nodeList.forEach(function(element, index) {
      if (index % 2 === 0) {
        element.classList.add('highlight');
      } else {
        element.classList.add('bold');
        element.addEventListener('click', function() {
          alert('Haz hecho clic en el elemento: ' + element.textContent);
        });
      }
    });
